# Solução – Exercícios da Aula 12

Aqui você encontra as respostas sugeridas para os exercícios de Markdown da Aula 12.

### 1. Sintaxe de Títulos
A sintaxe é:
- `# Título 1` (Maior)
- `## Título 2`
- `### Título 3` (Menor)

### 2. Links e Ênfase
Exemplo de frase formatada:
"Este é o meu **perfil oficial** no [GitHub](https://github.com/seu-usuario)."

### 3. Listas e Itens
Para a checklist:
- [x] Git
- [x] GitHub
- [x] Markdown

### 4. Blocos de Código
Para criar o bloco, usamos três crases (backticks) seguidas do nome da linguagem:
```bash
git clone https://link-do-repo.git
```
O Markdown foi avisado através das três crases no início e no fim do bloco.

### 5. Tabelas e Organização
Exemplo de tabela com alinhamento à direita na segunda coluna (`---:`):
| Comando | Função |
| :--- | ---: |
| `git init` | Inicia repo |
| `git commit` | Salva versão |
| `git push` | Sobe código |
