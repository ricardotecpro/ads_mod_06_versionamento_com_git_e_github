# Quiz 13

1. Qual menu do GitHub usamos para adicionar pessoas para trabalharem no projeto conosco?

    - [ ] Actions
    - [x] Settings > Collaborators
    - [ ] Pull Requests
    - [ ] Insights

2. Um "Colaborador" com acesso de escrita precisa fazer Fork do projeto para contribuir?

    - [ ] Sim, sempre
    - [x] Não, ele pode criar branches e dar push direto no repositório oficial
    - [ ] Apenas se o projeto for público
    - [ ] Apenas se ele usar Windows

3. Qual a diferença principal entre o dono (Owner) e um Colaborador?

    - [ ] Nenhuma, são iguais
    - [ ] O Colaborador não pode ver o código
    - [x] O Owner tem poder total, inclusive para deletar o repositório ou mudar configurações críticas
    - [ ] O Owner não pode commitar

4. O que acontece se seu colega der push na `main` e você tentar dar push sem atualizar seu PC antes?

    - [x] O Git rejeita seu push (erro non-fast-forward) e pede para você fazer pull primeiro
    - [ ] O Git apaga o que seu colega fez
    - [ ] O computador trava
    - [ ] O GitHub cria um repositório novo automaticamente

5. Qual comando baixa as atualizações do time do GitHub para o seu computador?

    - [ ] git download
    - [ ] git sync
    - [x] git pull
    - [ ] git push --reverse
